# 🍲 Cookbook Web App

A modern, full-stack web application for discovering and discussing recipes. Built as part of a university course project.

## ✨ Features

- 📝 browse recipes
- 🔍 Search recipes by titles
- 👍 Upvote recipes
- 💬 Comment on recipes
- ⚛️ Frontend in React
- 🌐 Backend in Node.js with RESTful API powered by Express
